My Story

I initially started my Web development training in years 10 years ago, but I had the fortunate luck being granted the Immigrant Visa to come to the US. So after gaining work experience in different fields I decided to return to my initial interest. My 1st 6 months in the US I worked as a construction worker, painting, demolition, windows ... that's when my friend offered me a job working for Sammy Sosa as a warehouse worker. During that period I obtained my forklift operation certificate, which lead me to be the warehouse supervisor and the operator of the steel machine that manufactures bundles of steel we used to ship to South America so the rest of the crew put these pieces together to build Anti-Disaster houses for low income families. After that I decided to join the restaurant field, I worked as a busboy, then as a cook, sous chef back in South Florida, and I ended up as kitchen supervisor in Sunport Airport when I moved to Albuquerque in 2017. After that I applied to Baillos and I worked there as a delivery driver and as an appliances installer; fridges, dishwashers, dryers, Tvs ...

My Work

HTML5

HTML is the 1st coding language I've learned, HTML, in full hypertext markup language, a formatting system for displaying material retrieved over the Internet. Each retrieval unit is known as a Web page (from World Wide Web), and such pages frequently contain hypertext links that allow related pages to be retrieved.

CSS

CSS is the 2nd language I learn to use and it is acronym of “Cascading Style Sheets ”. CSS is a computer language for laying out and structuring web pages (HTML or XML). This language contains coding elements and is composed of these “cascading style sheets” which are equally called CSS files.

Bootstrap 5

Bootstrap is the 3rd coding language I've learned to use and it is a free and open source front end development framework for the creation of websites and web apps. Its framework is built on HTML, CSS, and JavaScript (JS) to facilitate the development of responsive, mobile-first sites and apps

Node JS

Node. js (Node) is the open source development platform for executing JavaScript code server-side. I learned and because it is useful for developing applications that require a persistent connection from the browser to the server and is often used for real-time applications such as chat, news feeds and web push notifications.
